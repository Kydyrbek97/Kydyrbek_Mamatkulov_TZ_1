Linux commands 
Ctrl+Alt+T - команда открытия терминала

~(тильда) - символ домашней директории
/ - символ корневой директории

.. - обозначают предыдущие директорию
. - обозначает текущую директорию

1. ls(list) - команда для вывода всех директорий(папка) и файлов в текущей (рабочий) директории в терминале 
2. cd (change directory) -  команда для изменения (переходов) одной директории на другую cd <название директории>
3. mkdir (make directory) - создает указанную директорию в рабочей директории в терминале
4. touch - создает указанный файл
5. rm - удаление файла
6. rmdir - удаление директории
7. pwd (present working directory) - выводит путь до текущей директории
8. rm -rf - принудительного удаления
9. ls -a - выводит список как скрытых так и обычных файлов и директорий.
10. nano <path_to_file_> - откроется файл для редактирования в графическом интерфейсе * если нет файла, то nano попытаться создать новый файл.
11. cat <file_name>. - выводит содержимое файла в терминал 
12. echo - это команда для быстрой записи текстовой иныормации в какой либо файл.
    echo текст_для_записи > название_файла
13. mv - команда для переименовывания файла либо директории, а также для перемещения файлов или директории.
     Переименовывания: mv jonh john
                       mv <старое_название> <новое_название>
     Перемещение: mv john ~/Desktop
                  mv <file/directory> <directory>/
14.cp - копирование
